Contest Notifier

Contest Notifier is a basic Linux based desktop app to keep one informed with upcoming contests on TopCoder and CodeForces. The application shows the time of the contests in desired TimeZone and can be synced manually even behind a proxy connection.

TOJHelper

This plugin can be integrated with Sublime Text and is made with an intention to help competitive programmers in their practice on Timus Online Judge. The plugin has features to download the test cases for a particular question, run program on all of them and to submit the solution from the Sublime itself.

  • Project Link : TOJHelper

CodeForces ToolKit

A simple CodeForces Toolkit which allows the users to view stats of other user's and let's one see the questions done by other user. We hope such a toolkit can help those programmers who are new to competitive programmers and are looking for a point to start from.

  • Project Link : CodeForces Toolkit
